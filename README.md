# 🚀 AI-Powered Customer Support SaaS Platform

A modern AI-powered customer support SaaS platform built using Next.js, AI integration, authentication system, chatbot embedding, live preview system, and scalable SaaS architecture.

This platform helps businesses create and manage AI-powered customer support chatbots with a premium user experience.

# 📌 Overview

AI Customer Support SaaS Platform is a full-stack SaaS application that enables businesses to integrate AI customer support assistants into their websites.

The platform provides:

- AI chatbot management
- Chatbot embedding system
- Live chatbot preview
- Authentication system
- Modern dashboard
- Scalable SaaS architecture
- Responsive UI/UX

The project is designed with modern SaaS product standards and production-level architecture.
